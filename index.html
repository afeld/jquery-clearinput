jquery-clearinput by afeld

Clears the instructions from an HTML form input on focus, replacing the text if nothing is entered.

Usage

Download the plugin, then add the following to the <head> of your HTML:

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.6.4/jquery.min.js" type="text/javascript"></script>
<script src="jquery.clearinput.js" type="text/javascript"></script>
The default value can be specified as a param:
<input name="email" id="example1">

<script type="text/javascript">
  $(document).ready(function(){
    $("#example1").clearInput("Enter name");
  });
</script>
or as a data-empty-value attribute:
<input name="email" data-empty-value="Enter email" id="example2">

<script type="text/javascript">
  $(document).ready(function(){
    $("#example2").clearInput();
  });
</script>
It even works on contenteditable fields:
<div contenteditable="true" id="example3"></div>

<script type="text/javascript">
  $(document).ready(function(){
    $("#example3").clearInput("Type here");
  });
</script>

jquery-clearinput also adds an is-empty class to the element, when the field has the default value. You can have multiple clear-input fields on a single page, and they will not conflict. See sample.html for the very basic usage.

Requirements

Compatible with jQuery v1.2+.

See also

This plugin sets the usage text as the value for the input element. To use labels, see Doug Neiner's In-Field Labels plugin.
